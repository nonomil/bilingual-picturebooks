## 三、每课格式规范

### 3.1 中文课格式

```markdown
# 第N课 主题

## 📋 学习目标
- ...
- ...

## 🎬 第X页：Minecraft故事场景描述
Steve & Alex 在方块世界的冒险...
每个字展示：字形 → 配图 → 笔顺 → 组词

## ✏️ 练习
笔画描红 / 填空 / 配对 / 组词

## 🎯 挑战
小任务 / 密室逃脱 / 猜字谜

## 🎉 庆祝徽章 + 新字汇总 + 笔画顺序表

### ✨ 本课新字
| 字 | 拼音 | 笔画顺序 | 组词 |
|----|------|---------|------|
```

### 3.2 拼音课特殊格式

```markdown
# 第N课 拼音主题

## 📋 学习目标
- 认识...
- 掌握...

## 🎬 第X页：Minecraft拼音故事
- 拼音=读字的魔法工具（像Minecraft附魔）
- 每个拼音对应一种"魔法符号"
- 用已学过的汉字做拼读示范

## 🎵 练一练
- 通过情境场景练习发音
- 用节奏/重复/游戏强化记忆

## ✏️ 书写练习
- 四线格声调标注
```

### 3.3 英语课格式

```markdown
# Lesson N — Theme

## 📋 Learning Goals
- ...

## 🎬 Page X: Minecraft story scene
- Story-driven, speech bubbles
- Each word with picture
- Sight words highlighted

## 🎵 Song / Chant
- Sing-along to familiar tunes

## Game / Challenge
- Letter/word puzzles
- Sentence building
- Village quest
```
