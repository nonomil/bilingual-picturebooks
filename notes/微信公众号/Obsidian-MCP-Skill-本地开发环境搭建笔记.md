## 🔑 关键要点

1. **Obsidian 有 MCP** — 可以通过 MCP 协议与外部工具通信
2. **GitHub/Gitee 同步最简单** — 不需要 MCP，直接发链接就能同步
3. **MCP 是统一协议** — 把不同的数据通过统一接口整合在一起
4. **数据自由可控** — 可以单独控制某一篇文章同步或不同步
5. **大白话操作** — 把复杂的同步操作变成一句话让 AI 执行
6. **AI 辅助画图** — 用 OpenCode 等工具直接用大白话生成流程图

## 💡 核心优势

| 优势 | 说明 |
|------|------|
| 数据自主可控 | 数据牢牢掌握在自己手里，不会丢失 |
| 不依赖特定网站 | 不用担心网站关闭对自己的影响 |
| 批量处理 | 一分钟内完成对一套教材的剖析 |
| AI 赋能 | 复杂操作变自然语言指令 |
| 多端同步 | MCP 统一协议，自由控制每篇文章同步 |

## 🎯 应用场景

- 教材逻辑流程图批量生成
- 课文/文章自动分析为流程图
- 笔记多端自由同步
- AI 直接读写 Obsidian 笔记
