## Phonics: Light Integration (Sound Blocks 🔤)

### Philosophy
Full systematic phonics requires heavy input. For this course (light ESL, ~47 words by L4), we do **exposure-level phonics** — each lesson introduces ONE phoneme tied to the lesson vocabulary. Called "Sound Blocks" in-game (Minecraft sound note blocks).

### Format (per lesson)
```
🔤 Sound Block: /æ/ as in "apple, cat, hat"
   Steve hits a note block → "a-a-a-apple!"
   
   Can you say?
   c-a-t → cat
   h-a-t → hat
   m-a-p → map
```

|--------|---------|---------------|--------------|

### Extension Lessons
Each extension reinforces the base lesson's phonics rule + adds 1-2 CVC words for decoding practice.
