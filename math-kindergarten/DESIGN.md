# 方块数学 V2 — 设计规范

## 基本信息
- **目标读者**: 5-7岁，幼小衔接
- **主题**: Minecraft 方块世界冒险
- **教学方法**: 新加坡 CPA（Concrete→Pictorial→Abstract）+ 苏格拉底提问 + 冒险关卡制

## 核心设计理念（融合3本参考书）

### 来自「我的世界·数学思维游戏」
- **冒险关卡制**：每章=一个冒险任务，帮Steve/Alex解决问题
- **游戏类型多样化**：连线、涂画、找茬、排序、迷宫、贴纸、情景
- **7大数学思维种子**：归纳整理、顺序概念、等价转换、抽象化、具体化、逆向思维、对称性
- **难度曲线**：整体由易到难，难中有易（抗挫力），易中有难（防骄傲）

### 来自「All the Maths You Need to Know by Age 7」(Usborne)
- **生活化场景**：每个概念从日常生活引入（数苹果、分蛋糕、看时钟）
- **角色贯穿**：固定角色串起故事，孩子有亲切感
- **互动翻翻形式**：一问一答、猜猜看、找一找
- **章节结构**：What is maths? → Numbers → Adding → Shapes → Measuring → Patterns

### 来自「新加坡数学」
- **CPA三步法**：Concrete(实物) → Pictorial(图象) → Abstract(符号)
- **Number Bonds(数的朋友)**：5=2+3=1+4，从拆分组合理解数
- **模型画法(Model Drawing)**：用方块图表示数量关系

## 核心原则
1. **文字极少，图片主导** — 每页≤40字，图片占页面70%+
2. **先玩后学** — 每章先讲故事/游戏，再引入概念，最后练习
3. **每页一个知识点** — 不贪多，一步一阶
4. **大量活动页** — 每章活动页≥40%（涂色、连线、迷宫、找不同、贴纸）

## 每章结构（冒险关卡制）

每章20-24页，结构如下：

```
🎮 关卡标题页（1页）
📖 故事导入：Steve遇到问题（2-3页）
🧱 Concrete：用Minecraft实物操作（3-4页）
🖼️ Pictorial：看图理解（2-3页）
🔢 Abstract：数字与符号（2-3页）
✏️ 活动时间：游戏/练习（5-8页）
  - 涂色页（1-2页）
  - 连线/匹配（1-2页）
  - 迷宫/找不同（1-2页）
  - 贴纸/剪贴（1-2页）
💡 挑战关：Boss战（1-2页）
🏠 回家休息：小结+下章预告（1页）
```

## 图片风格规范
- **风格**: Minecraft像素风 + 水彩儿童绘本温暖色调
- **角色**: 
  - Steve（蓝色衬衫，棕色裤子，棕色头发）
  - Alex（绿色衬衫，橙色头发）
  - 动物：小羊、小猪、鸡、牛、猫、狼（反派）
- **色调**: 明亮、温暖、高饱和度，绿色草地+蓝色天空为主基调
- **构图**: 简洁，主体突出，留白充足
- **活动页风格**: 黑白线稿为主，方便涂色/书写
- **尺寸**: 正方形 1:1（适合手机/平板阅读）

## 活动页设计规范

### 涂色页
- 黑白线稿，Minecraft风格
- 按数字涂色（1=红色, 2=蓝色...）
- 涂完能看到隐藏图案

### 连线/匹配页
- 左边数字，右边对应数量的物品
- 或左边算式，右边答案
- 连线用虚线引导

### 迷宫页
- 方块世界主题迷宫
- 沿着正确答案的路径走（如：沿1-10的顺序走）

### 找一找页
- 繁忙的Minecraft场景
- 找出指定数量的物品
- 找不同（两张图有5处不同）

### 数一数页
- 散落的物品图
- 写出数量
- 圈出多/少的组
