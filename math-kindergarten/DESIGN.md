# 方块数学 V2 — 设计规范

## 基本信息
- **目标读者**: 5-7岁，幼小衔接
- **主题**: Minecraft 方块世界冒险
- **教学方法**: 新加坡 CPA（Concrete→Pictorial→Abstract）+ 苏格拉底提问 + 冒险关卡制
- **总册数**: 上册（本册，12章）+ 下册（乘除法、分数、时间、金钱等）

## 核心设计理念（融合参考书优势）

### 来自「我的世界·数学思维游戏」（Mojang 官方授权，6册）
- **冒险关卡制**：每章 = 一个冒险任务，帮 Steve/Alex 解决问题
- **游戏类型多样化**：连线、涂画、找茬、排序、迷宫、贴纸、情景模拟
- **7大数学思维种子**：归纳整理、顺序概念、等价转换、抽象化、具体化、逆向思维、对称性
- **人大附中分级进阶**：难度曲线科学，整体由易到难，难中有易（抗挫力），易中有难（防骄傲）
- **覆盖领域**：数与代数、图形与几何、测量、统计

### 来自「All the Maths You Need to Know by Age 7」（Usborne）
- **生活化场景**：每个概念从日常生活引入（数苹果、分蛋糕、看时钟）
- **角色贯穿**：固定角色串起故事，孩子有亲切感（虫子角色）
- **互动翻翻形式**：一问一答、猜猜看、找一找，激发好奇心
- **章节结构清晰**：What is maths? → Numbers → Adding → Shapes → Measuring → Patterns
- **语言极简**：用最少的文字讲清概念，大量留白给图片

### 来自「新加坡数学」
- **CPA三步法**：Concrete(实物) → Pictorial(图象) → Abstract(符号)
- **Number Bonds(数的朋友)**：5=2+3=1+4，从拆分组合理解数
- **模型画法(Model Drawing)**：用方块图表示数量关系

## 核心原则

1. **文字极少，图片主导** — 每页≤40字，图片占页面70%+
2. **先玩后学** — 每章先讲故事/游戏，再引入概念，最后练习
3. **每页一个知识点** — 不贪多，一步一阶
4. **大量活动页** — 每章活动页≥40%（涂色、连线、迷宫、找不同、贴纸）
5. **互动感** — 一问一答形式，让孩子主动思考而非被动接受
6. **生活化** — Minecraft场景即生活场景（交易、建造、挖矿、种地）

## 教材章节结构

每章一个独立 MD 文件，结构如下：

```
# 第X课 课题名称

## 📋 学习目标
- 目标1
- 目标2
- 目标3

---

## 一、故事导入（2-3页配图）
场景描述 + Steve/Alex遇到问题
→ 引发好奇心："你能帮他吗？"

## 二、知识讲解（5-8页配图）
按CPA三步法展开：
  Concrete → Minecraft实物操作
  Pictorial → 图象理解
  Abstract → 数字符号

每个小节 = 1个知识点 + 1张配图

## 三、课堂练习（5-8页配图）
活动类型轮换：
  - 涂色页（按数字涂色）
  - 连线/匹配页
  - 迷宫页
  - 找不同页
  - 数一数页
  - 描红/写数字页

## 四、Boss挑战（1-2页配图）
综合运用本章知识解决问题

## 五、本课小结
✅ 学会了什么
> 下一课预告
```

## 图片风格规范
- **风格**: Minecraft像素风 + 水彩儿童绘本温暖色调
- **角色**: Steve（蓝色衬衫）、Alex（绿色衬衫）、动物（羊、猪、鸡、牛、猫）、敌对生物（僵尸、苦力怕、末影人）
- **色调**: 明亮、温暖、高饱和度，绿色草地+蓝色天空为主基调
- **构图**: 简洁，主体突出，留白充足
- **活动页**: 黑白线稿为主，方便涂色/书写
- **尺寸**: 正方形 1:1
