在基础课里，我们学过：

- 加法：十位加十位，个位加个位
- 减法：十位减十位，个位减个位

Minecraft里，Alex要帮村庄建围墙。  
每一层先用整十块石头，最后再加一点零头。

**Steve：** 你还记得怎么加吗？  
**Alex：** 个位对个位，十位对十位！  
**Steve：** 那怎么减呢？  
**Alex：** 也要对齐，再分别算。

### 一起想一想
1. 23 里面有几个十和几个一？  
2. 41 里面的 4 在十位还是个位？  
3. 32 + 15，应该先看哪一位？

### 小复习
**例1：21 + 16**

21 = 2个十和1个一  
16 = 1个十和6个一  

十位：2个十 + 1个十 = 3个十  
个位：1个一 + 6个一 = 7个一  

所以：**21 + 16 = 37**

**例2：48 - 26**

48 = 4个十和8个一  
26 = 2个十和6个一  

十位：4个十 - 2个十 = 2个十  
个位：8个一 - 6个一 = 2个一  

所以：**48 - 26 = 22**

### 竖式看一看
```text
  21
+ 16
----
  37
```

```text
  48
- 26
----
  22
```

## 👋 第三页：小拓展
<img src="./img/chapter-14/page-03.png" width="480" alt="Steve教Alex用竖式计算石头剩余数量，练习两位数减法"/>

围墙仓库里原来有 **67** 块石头。  
Alex搬走了 **25** 块。  
还剩多少块？

**Steve：** 这是加法还是减法？  
**Alex：** 搬走了，就是减法！  
**Steve：** 哪一位先算？  
**Alex：** 个位减个位，十位减十位。

### 一步一步来
67 = 6个十和7个一  
25 = 2个十和5个一  

十位：6个十 - 2个十 = 4个十  
个位：7个一 - 5个一 = 2个一  

所以：**67 - 25 = 42**

### 竖式
```text
  67
- 25
----
  42
```

### 看一看，想一想
1. 76 - 34 = □  
2. 58 - 16 = □  
3. 95 - 42 = □

### 答案
1. 76 - 34 = **42**  
2. 58 - 16 = **42**  
3. 95 - 42 = **53**

### ❌常见误解：错误 vs 正确

**错误：没对齐**
```text
  32
+14
----
 46
```

**正确：要对齐**
```text
  32
+ 14
----
  46
```

**错误：十位和个位乱加**  
把 3 和 4 加，2 和 1 加。这样不对。

**正确：按位置算**  
- 十位对十位  
- 个位对个位

### 终极挑战
**问题1：** 前两层一共用了多少块？  
30 + 16 = □

**问题2：** 前两层加上后来拿来的，一共有多少块石头可用？  
(30 + 16) + 21 = □

**问题3：** 用完后还剩18块，说明一开始总共有多少块？  
你能从上面找到答案吗？

### 一起做
**问题1：**  
30 + 16 = **46**

**问题2：**  
46 + 21 = **67**

**问题3：**  
一共有 **67** 块，最后剩 **18** 块没用。  
说明用掉了一部分。  
这里先知道：**总数是67块。**

### 再挑战
从 67 块里，剩下 18 块。  
那用了多少块？

67 - 18 = **49**

## 📖 小词典
- **十位**：表示几个十的位置
- **个位**：表示几个一的位置
- **竖式**：把数字上下对齐来计算的方法
- **一共**：合起来的总数
- **还剩**：用掉一些以后留下来的数量
- **对齐**：十位和十位放一起，个位和个位放一起
