# Math Kindergarten — English Edition

Bilingual adaptation of the Math Kindergarten (方块数学 V2) picture book series.

## Structure

- `lesson-XX-*.md` — Base lesson (teaches core concept)
- `lesson-XX-extension.md` — Extension lesson (reinforces with new context)

## Scope (Pilot)

Lessons 1-6 cover:
1. Counting 1–10
2. Numbers 11–20
3. Comparing (>, <, =)
4. Addition Within 5 (+ Number Bonds)
5. Addition Within 10 (+ Make Ten)
6. Subtraction Within 5

## Content Notes

- All illustrations shared with Chinese edition (`img/` symlinks to `../chapters/img/`)
- Curriculum standards aligned to **US Common Core K** (K.CC, K.OA, K.NBT, K.MD)
- Same Steve & Alex Minecraft storyline
- Number Bonds (*Part-Part-Whole*) added as pedagogical enhancement
