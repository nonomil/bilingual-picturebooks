# 第23课 拓展篇：悯农 + 写诗工坊

## 📋 学习目标
- 学习第四首古诗《悯农》
- 了解古诗的押韵
- 尝试自己写诗

---

## 🎬 第一页：悯农

> "学了三首诗，再加一首——《悯农》。这首诗告诉你为什么要爱惜食物。"

```
   🌾 《悯农》 [唐] 李绅
   
   锄禾日当午，
   汗滴禾下土。
   谁知盘中餐，
   粒粒皆辛苦。
```

**诗意解读：** 农民在正午的烈日下锄地，汗水滴在禾苗下面的泥土里。谁知道碗里的每一粒饭，都是农民辛苦种出来的？

> "你在第15课学了'米'和'饭'——现在知道每一粒米是怎么来的了。"

---

## 🎬 第二页：押韵的秘密

> "为什么读诗感觉像唱歌？因为诗有'押韵'——句尾的字发音相似！"

```
   🔔 找押韵：
   
   《静夜思》：光(guāng) — 霜(shuāng) — 乡(xiāng)
   最后都有 āng 的声音！
   
   《春晓》：晓(xiǎo) — 鸟(niǎo) — 少(shǎo)
   最后都有 ǎo 的声音！
```

Steve试着押韵："大(dà)——花(huā)——家(jiā)，都有ā！"

Alex也试试："星(xīng)——明(míng)——听(tīng)，都有īng！"

---

## 📝 练习 — 写诗

用押韵的方法写一首四行诗：

```
   提示：找到一个韵脚（比如 āng），
   每一行的最后一个字都用这个韵。
   
   例子（āng韵）：
   天上有太___，
   地上有花___。
   小鸟在歌___，
   世界真美___。
```
