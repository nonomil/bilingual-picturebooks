# 第9课 声母王国（上）

## 📋 学习目标
- 认识前 8 个声母：**b p m f d t n l**
- 理解"声母 + 韵母"的拼读规则
- 用声母和已学单韵母拼出简单音节
- 区分送气音（p t）和不送气音（b d）

**拼音阶段不引入新汉字，用已学 58 字做拼读练习**

```

   p — 嘴唇闭紧，用力喷出气

**b**（蓝色铠甲）先站出来："听好了——b！"

> "嘴唇闭紧，突然爆开。像放了一个小炮仗：b！"

   嘴唇一碰 b b b。

> "跟 b 很像，但要喷出气来！手心放在嘴前，感觉到气了吗？"

Alex 把手放在嘴前："p——感觉到了！好多气！"

```
   p 的口诀：

   💨 b 和 p 的区别：
   b = 不出气（不送气）😐
   p = 喷出气（送气）  💨
```

**m**（粉色铠甲）温柔地说："m——嘴唇闭上，鼻子出气。"

**f**（绿色铠甲）最后说："f——上牙轻轻咬下唇。"

> "像风吹过：f——f——"

## 🎬 第三页：b 和 p — 炮仗兄弟

b 将军和 p 大副站在一起。

> "我们两个最容易搞混——但其实有秘诀！"

```
   b vs p 大挑战：
   
   同样的口型：
   ① 嘴唇闭紧
   ② 突然打开
   
   不同之处：
   b → 喉咙不振动，不喷气（轻轻的）
   p → 喉咙不振动，用力喷气！
```

> "试试看：一张薄纸放在嘴前。"

Steve 拿着一张纸，放在嘴前。

> "念 b——" 纸不动。
> "念 p——" 纸飞起来！📄💨

```
   b 不吹纸 📄   vs   p 吹动纸 📄💨
```

> "现在用 a 来拼读！"

```
   b + a = ba
   bā bá bǎ bà
   
   bā — 八（八个）
   bǎ — 把（一把尺子）
   bà — 爸（爸爸）
   
   p + a = pa
   pā pá pǎ pà
   
   pā — 趴（趴下）
   pá — 爬（爬山）
   pà — 怕（害怕）
```

Alex 发现了："同一个 a，b 和 p 拼出来是完全不同的字！"

> "没错！这就是声母的魔力——同样的韵母，不同的声母，意思完全不同！"

## 🎬 第六页：d t n l 大练兵

四剑客各自展示它们的拼读本领：

```
   🗡️ d 的拼读：
   d + a = da    dà — 大（大人）
   d + e = de    dé — 得（得到）
   d + i = di    dì — 地（大地）
   d + u = du    dú — 读（读书）
   
   🗡️ t 的拼读：
   t + a = ta    tā — 他（他们）
   t + i = ti    tī — 梯（楼梯）
   t + u = tu    tú — 图（图画）
   
   🗡️ n 的拼读：
   n + a = na    nǎ — 哪（哪里）
   n + i = ni    nǐ — 你（你好）
   n + u = nu    nǔ — 努（努力）
   n + ü = nü    nǚ — 女（女生）
   
   🗡️ l 的拼读：
   l + a = la    lā — 拉（拉手）
   l + e = le    lè — 乐（快乐）
   l + i = li    lì — 力（力气）
   l + u = lu    lù — 路（走路）
   l + ü = lü    lǜ — 绿（绿色）
```

> "注意！n 和 l 都可以跟 ü 拼！"

```
   n + ǚ = nǚ（女）    l + ǜ = lǜ（绿）
```

> "咦，n 和 l 听起来有点像……"Alex 皱眉。

> "对！n 是从鼻子出气，l 是从舌头两边出气。再来一次："

```
   n——（捏住鼻子发不出来！因为气从鼻子走）
   l——（捏住鼻子也能发！因为气从舌边走）
```

> "这就是区分 n 和 l 的秘诀——捏住鼻子！"

Steve 试了试："真的！n 发不出来，l 可以！"

```
   🎯 声母(8) × 韵母(6) = 拼音大世界！
```

> "来玩对对碰！一声母碰一韵母——看能拼出什么！"

```
   b + a = ba     p + a = pa     m + a = ma     f + a = fa
   b + o = bo     p + o = po     m + o = mo     f + o = fo
   b + i = bi     p + i = pi     m + i = mi
   b + u = bu     p + u = pu     m + u = mu     f + u = fu
   
   d + a = da     t + a = ta     n + a = na     l + a = la

   d + i = di     t + i = ti     n + i = ni     l + i = li

### ❌常见误解

| ❌ 错误发音/写法 | ✅ 正确发音/写法 |
|-------|-------|
| b和d分不清（"b像6，d反6"） | b=右耳b，圆圈在右；d=左耳d，圆圈在左 |
| p和q分不清（"p像9，q反9"） | p=泼水p，圆圈在右向下；q=气球q，圆圈在左向下 |
| zh/ch/sh 读成 j/q/x | 舌尖翘起（翘舌音），不是舌面贴硬腭 |
| 前鼻音和后鼻音不分 | an≠ang，舌头位置不同（前鼻音舌尖顶上齿，后鼻音舌根抬起） |

🧠 想一想
1. **观察推理**：b、p、m、f 都用嘴唇发音，d、t、n、l 用舌尖。你还能找到其他发音位置的规律吗？
2. **反事实**：如果拼音里没有声调（一、二、三、四声），"妈妈骂马"这4个字还能分得清吗？
