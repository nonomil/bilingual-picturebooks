- 学会后 3 个单韵母：**i u ü**

**拼音阶段不引入新汉字，用已学 58 字做拼读练习**

## 🎬 第二页：i — 牙齿对齐

💚 绿色精灵 **i** 跳到 Steve 面前。

```
   i 的读法：
   
   ① 牙齿对齐
   ② 嘴角向两边咧开
   ③ 舌尖抵住下牙
   ④ 发出声音：衣——
```

> "像穿衣服：衣——"

> "ī——" Steve 发出声音。

i 精灵高兴地旋转，书页上浮现出：

```
   i 的四声：
   
   ī  —  īīī   （一声平：衣服的衣 👔）
   í  —  ííí   （二声扬：阿姨的姨 👩）
   ǐ  —  ǐǐǐ   （三声拐弯：椅子的椅 🪑）
   ì  —  ììì   （四声降：意思的意 💡）
```

> "注意！i 戴上声调帽子时，上面的点不见了！"

```
   i + ˉ = ī  （帽子代替了点）
   i + ˊ = í
   i + ˇ = ǐ
   i + ˋ = ì
```

> "就像你戴帽子的时候，要把头发塞进帽子里！"i 精灵俏皮地说。

```
   📝 i 的小秘密：
   小 i 有礼貌，标调就脱帽
   小点不见了，帽子头上戴
```

Alex 试着拼读学过的字：

```
   笔 (bǐ) — ǐ！第三声！
   你 (nǐ) — ǐ！第三声！
   地 (dì) — ì！第四声！
```

## 🎬 第三页：u — 嘴巴突出

💜 紫色精灵 **u** 滚了过来。

```
   u 的读法：
   
   ① 嘴巴向前突出，圆圆的小洞
   ② 舌头向后缩
   ③ 发出声音：乌——
```

> "像乌鸦叫：乌——乌——"

> "ū——" Alex 噘起嘴。

```
   u 的四声：
   
   ū  —  ūūū   （一声平：乌鸦的乌 🐦‍⬛）
   ú  —  úúú   （二声扬：无用的无 🚫）
   ǔ  —  ǔǔǔ   （三声拐弯：五个的五 🖐️）
   ù  —  ùùù   （四声降：大雾的雾 🌫️）
```

Steve 试着拼：

```
   书 (shū) — ū！第一声！
   读 (dú) — ú！第二声！
   不 (bú) — ú！第二声！
   木 (mù) — ù！第四声！
```

> "原来这么多字都有 u！"

## 🎬 第四页：ü — 嘴巴圆圆像吹笛

最后，🧡 橙色精灵 **ü** 优雅地飘过来。

```
   ü 的读法：
   
   ① 嘴巴圆圆撮起来
   ② 像吹笛子一样
   ③ 发出声音：鱼——
```

> "像小鱼吐泡泡：ü——"

> "ǖ——" 两人一起撮起嘴。

```
   ü 的四声：
   
   ǖ  —  ǖǖǖ   （一声平：淤泥的淤 🏞️）
   ǘ  —  ǘǘǘ   （二声扬：小鱼的鱼 🐟）
   ǚ  —  ǚǚǚ   （三声拐弯：下雨的雨 🌧️）
   ǜ  —  ǜǜǜ   （四声降：玉石的玉 💎）
```

> "注意！ü 跟 i 不一样——ü 的两点永远在，即使戴帽子！"

```
   i 戴帽子 → 点没了：ī í ǐ ì
   ü 戴帽子 → 点还在：ǖ ǘ ǚ ǜ
```

Alex 发现了：

```
   云 (yún) — ǘn！第二声！
   女 (nǚ) — ǚ！第三声！
   绿 (lǜ) — ǜ！第四声！
```

> "ü 就像一条小鱼，永远睁着两只眼睛！"

## 📝 练习

### 一、读一读，连一连

| 韵母 | 口型 | 代表字 |
|------|------|--------|
| i | 张大嘴巴 | 笔 bǐ |
| u | 嘴巴圆圆，撮起来 | 书 shū |
| ü | 牙齿对齐 | 雨 yǔ |
| a | 嘴巴突出 | 大 dà |

### 二、补全拼音

```
   笔 → b___ （韵母是 i，第三声）
   书 → sh___ （韵母是 u，第一声）
   雨 → ___ǔ （声母是 y，韵母是 ü）
   地 → d___ （韵母是 i，第四声）
   木 → m___ （韵母是 u，第四声）
   女 → n___ （韵母是 ü，第三声）
```

### 三、声调练习

写出 i、u、ü 的四声：

```
   i：___ ___ ___ ___
   u：___ ___ ___ ___
   ü：___ ___ ___ ___
```

### 四、读儿歌

```
   i i i，牙齿对齐衣衣衣，
   u u u，嘴巴突出乌乌乌，
   ü ü ü，撮嘴吹笛鱼鱼鱼。
   
   ɑ o e，i u ü，
```
