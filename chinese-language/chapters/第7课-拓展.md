# 第7课 拓展篇：四声大冒险

## 📋 学习目标
- 巩固单韵母 a o e + 四声
- 学会标注声调——给拼音加帽子
- 用拼音拼读已学汉字
- 区分同音不同调的字

---

## 🎬 第一页：拼音石的秘密

从拼音小镇回来后，Steve 和 Alex 各自握着自己的拼音石。

第二天早上一醒来，Steve 发现拼音石在发光——

> 💎 "来找我们……来声调山……"

> Alex 的拼音石也在发光！"它叫我们去声调山！"

两人跟着拼音石的指引，穿过村庄，来到一座奇怪的山前。

这座山的形状跟上次书上画的一模一样：

```
         ˉ 平顶
       /      \
      / ˊ      \ ˋ
     /   上坡    \ 下坡
    /    ˇ       \___
   /    山谷        \
  /                  \
```

> "声调山！"他们异口同声。

山脚下立着一块石碑：

```
   🏔️ 声调山
   
   要到达山顶，
   必须通过 4 个关卡：
   
   第一关：一声平 —— 走平路
   第二关：二声扬 —— 爬山坡
   第三关：三声拐弯 —— 过山谷
   第四关：四声降 —— 滑下去
```

> "Are you ready?" Alex 用刚学的英语问。

> "Ready!" Steve 握紧拼音石。

## 🎬 第二页：第一关 — 一声平

第一段路是一条笔直的石板路。路牌写着：

```
   第一关：ˉ 一声平
   
   规则：读出所有「一声」的字才能通过
   一声 = 高高平平的声音
```

路的两旁长着发出 ā—— 声音的花。

```
   🌸 ā 花  🌸 ā 花  🌸 ā 花
```

> "一声——平！"Steve 踏上去。

每踩一块石板，就有一个带着一声的字跳出来：

```
   石板1：花 (huā) 🌸 — ā！
   石板2：天 (tiān) ☁️ — ān！
   石板3：山 (shān) ⛰️ — ān！
   石板4：心 (xīn) ❤️ — īn！
   石板5：妈 (mā) 👩 — ā！
```

两人一边走一边念。每念对一个字，石板就变成金色。

> "huā——tiān——shān——xīn——mā——"

走到尽头，一块金色石板升起，上面写着：

```
   ✨ 一声口诀：
   一声高高平又平，
   就像飞机云中行。
```

> "通关！"第一关的守卫精灵——红色 a 精灵出现了。"你们学得真好！"

## 🎬 第三页：第二关 — 二声扬

接着是一条向上的山路。路牌写着：

```
   第二关：ˊ 二声扬
   
   规则：读出所有「二声」的字才能爬上去
   二声 = 从低往高升
```

> "上坡了！"Alex 开始往上爬。

路边的树上挂着发出 á—— 声音的果子。

每爬一级台阶，就有一个字等着：

```
   台阶1：人 (rén) 🧍 — én！
   台阶2：虫 (chóng) 🐛 — óng！
   台阶3：学 (xué) 📚 — üé！
   台阶4：同 (tóng) 👫 — óng！
   台阶5：鹅 (é) 🦢 — é！
```

> "rén——chóng——xué——tóng——é——"

声音像爬山一样，从低到高。台阶变成银色。

走到坡顶，银色石碑上写着：

```
   ✨ 二声口诀：
   二声爬坡往上升，
   就像上山脚步轻。
```

黄色 o 精灵在坡顶等着："爬得不错！"

## 🎬 第四页：第三关 — 三声拐弯

第三关是一个 V 形的山谷。先进去，再出来。路牌写着：

```
   第三关：ˇ 三声拐弯
   
   规则：读出所有「三声」的字才能过山谷
   三声 = 先降下去再升上来
```

> "先下、后上——像过山车！"Steve 深吸一口气。

山谷里的石头发出 ǎ—— 的回音。每走一步：

```
   石阶1：我 (wǒ) 🙋 — ǒ！
   石阶2：火 (huǒ) 🔥 — uǒ！
   石阶3：水 (shuǐ) 💧 — uǐ！
   石阶4：小 (xiǎo) 👶 — iǎo！
   石阶5：鸟 (niǎo) 🐦 — iǎo！
```

> "wǒ——huǒ——shuǐ——xiǎo——niǎo——"

声音先下去再上来，像坐过山车。石阶变成铜色。

走到谷口，铜色石碑上写着：

```
   ✨ 三声口诀：
   三声先降再升起，
   就像小船水中行。
```

蓝色 e 精灵在山谷出口等着："拐弯成功！"

## 🎬 第五页：第四关 — 四声降

最后一段是一条下坡滑道。路牌写着：

```
   第四关：ˋ 四声降
   
   规则：读出所有「四声」的字才能滑下去
   四声 = 从高往低降
```

> "下坡啦！"Alex 欢呼。

滑道两旁的石柱发出 à—— 的轰鸣。每滑一段：

```
   滑道1：大 (dà) 🐘 — à！
   滑道2：下 (xià) ⬇️ — ià！
   滑道3：爸 (bà) 👨 — à！
   滑道4：地 (dì) 🌍 — ì！
   滑道5：叶 (yè) 🍃 — è！
```

> "dà——xià——bà——dì——yè——"

声音从高往低，像石头滚下山。滑道变成钻石色。

滑到底部，钻石石碑上写着：

```
   ✨ 四声口诀：
   四声高处向下落，
   就像石头滚下坡。
```

三个精灵——🔴a 🟡o 🔵e——在终点一起等着！

## 📝 练习

### 一、听音标调

听大人念下面的字，标出声调符号：

```
   ma  → m__   （妈 / 马 / 骂 ？听声音判断）
   da  → d__   （大 / 打 / 搭 ？）
   wo  → w__   （我 / 握 / 窝 ？）
   huo → hu__  （火 / 活 / 货 ？）
```

### 二、读一读，分一分

把下面的字按声调分组：

```
   花 天 山 人 虫 学 我 火 水 大 下 爸
    
   一声(ˉ)：___ ___ ___
   二声(ˊ)：___ ___ ___
   三声(ˇ)：___ ___ ___
   四声(ˋ)：___ ___ ___
```

### 三、补充声调口诀

```
   一声高高___又___，
   二声爬坡往___升，
   三声先___再___起，
   四声高处向___落。
```

### 四、拼一拼

把声母和韵母拼起来，加上声调：

```
   h + uā → huā (花)  🌸

   d + à  → ___  (大)  🐘
   w + ǒ  → ___  (我)  🙋
   sh + ān → ___ (山)  ⛰️
   r + én → ___ (人)  🧍
   x + iǎo → ___ (小) 👶
```

## 📊 拓展小结

声调技能：
- [ ] 一声 ˉ：高高平又平（花 天 山 妈 心）
- [ ] 二声 ˊ：爬坡往上升（人 虫 学 同 鹅）
- [ ] 三声 ˇ：先降再升起（我 火 水 小 鸟）
- [ ] 四声 ˋ：高处向下落（大 下 爸 地 叶）

口诀：
- [ ] 一声平，二声扬，三声拐弯，四声降

### ❌常见误解

| ❌ 错误发音/写法 | ✅ 正确发音/写法 |
|-------|-------|
| b和d分不清（"b像6，d反6"） | b=右耳b，圆圈在右；d=左耳d，圆圈在左 |
| p和q分不清（"p像9，q反9"） | p=泼水p，圆圈在右向下；q=气球q，圆圈在左向下 |
| zh/ch/sh 读成 j/q/x | 舌尖翘起（翘舌音），不是舌面贴硬腭 |
| 前鼻音和后鼻音不分 | an≠ang，舌头位置不同（前鼻音舌尖顶上齿，后鼻音舌根抬起） |

🧠 想一想
1. **观察推理**：b、p、m、f 都用嘴唇发音，d、t、n、l 用舌尖。你还能找到其他发音位置的规律吗？
2. **反事实**：如果拼音里没有声调（一、二、三、四声），"妈妈骂马"这4个字还能分得清吗？
