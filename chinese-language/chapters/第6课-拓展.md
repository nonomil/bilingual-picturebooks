# 第6课 拓展篇：方块学校的奇妙一天

## 📋 学习目标
- 巩固第6课 9 个字：老师同学书包本书笔尺
- 新增字：**校 写 读**（+3字）
- 学会描述学校生活
- 掌握"校"的结构（木+交），理解"写""读"的部首

**累计识字：58字**

---

## 🎬 第一页：方块学校的秘密

开心学校已经上了好几天课。Steve 和 Alex 越来越喜欢这里。

> "这座**学校**真特别。"Steve 说。

**校** [xiào] (10画)

> 左边"木"（木头建筑），右边"交"（交流知识）——木头房子里交流学问的地方。

```
笔画顺序：
①一(横)②一(竖)③丿(撇)④丶(捺)⑤丶(点)⑥一(横)⑦丿(撇)⑧丶(点)⑨丿(撇)⑩乀(捺)

写成口诀：木字旁，交字边，木房子里学知识
```

**组词：** 学校(xué xiào)、校园(xiào yuán)
**句子：** 我去学校上学。

> "等一下——那扇门后面是什么？"Alex 指着走廊尽头一扇彩色的门。

老师说："那是**方块图书馆**。走，带你们去看看！"

## 🎬 第二页：方块图书馆

推开门——哇！

满墙都是书架，每一格都放着五颜六色的方块书。有的方块会发光，有的会自己翻开。

> "这里的每一本书都藏着一个小世界。"老师说。

Steve 拿起一本红色的方块书。书自动打开了——

书页上跳出一个小人，开始讲故事：

> "很久很久以前，有一个方块王国……"

> "太神奇了！"Alex 喊道。"我也要**读**一本书！"

**读** [dú] (10画)

> 左边"讠"（言字旁，说话），右边"卖"——把文字说出来就是读。

```
笔画顺序：
①丶(点)②一(横折提)③一(横)④一(竖)⑤丶(点)⑥一(横钩)⑦一(横)⑧一(竖)⑨一(横钩)⑩丶(点)

写成口诀：言字旁加卖字边，读出声音在嘴边
```

**组词：** 读书(dú shū)、阅读(yuè dú)
**句子：** 我喜欢读书。

```

---

> 【标A: 语文课标一上·识字与写字·认识常用汉字（象形字→楷体）】

### ❌常见误解

| ❌ 错误写法/理解 | ✅ 正确写法/理解 |
|-------|-------|
| "日"写成"目"（中间多一横） | 日=太阳，中间一横，不是两横 |
| "山"写成三竖一样高 | 中间一竖最高，两边的低 |
| "水"的笔画随便写 | 笔顺：竖钩 → 横撇 → 撇 → 捺 |
| 把象形字当画看，不记字形 | 象形字是"从画变来的字"，要记住现在的样子 |

🧠 想一想
1. **观察推理**：为什么"日"里面只有一横而不是两横？（提示：太阳只有一个）
2. **反事实**：如果古人把"山"画成三座一样高的山峰，现在的"山"字会是什么样子？

Alex 拿起一本蓝色的书，书里跳出小鲸鱼在方块海里游来游去。

> "我在**读**关于大海的书！"她说。

## 🎬 第三页：魔法写字板

从图书馆出来，老师带大家来到另一间教室。

> "这是**写字教室**！"老师说。

教室里没有黑板——整面墙就是一个巨大的发光板。旁边挂着大大小小的笔。

> "在这面墙上，用这支笔——你的字会活过来！"

**写** [xiě] (5画)

> 上面是"冖"（秃宝盖），下面是"与"——用笔在纸上留下痕迹。

```
笔画顺序：
①丶(点)②一(横钩)③一(横)④一(竖折折钩)⑤一(横)

写成口诀：秃宝盖下加个与，纸上写字要用心
```

**组词：** 写字(xiě zì)、书写(shū xiě)
**句子：** 我用笔写字。

老师拿起一支大笔，在发光墙上写了一个"大"字。

```
   大
```

字刚写完，"大"字就从墙上跳出来，变成了一个巨大的发光方块，在地上蹦了两下！

> "哇——！"全班都叫了起来。

## 🎬 第四页：大家一起写

> "谁来试试？"老师问。

小方第一个举手。他拿起笔，在墙上写下：

```
   山
```

"山"字变成了一座小小的方块山，上面还有白云飘过！

同学们一个接一个：

```
   水 → 变成一条蓝色方块小河，哗哗流 🌊
   日 → 变成一个小太阳，在天上发光 ☀️
   火 → 变成一簇红色火焰，温暖地跳 🔥
```

Steve 走上去。他想了想——在墙上写了一个字：

```
   家
```

一个温馨的小房子出现了：有门、有窗、有烟囱，烟囱里飘出炊烟。

> "这是我家！"Steve 笑了。

Alex 接着写：

```
   爱
```

一颗大大的红心从墙上飞起来，飘到天花板上闪闪发光。

> "有爱的地方就是家。"Alex 说。

老师和同学们鼓掌。

## 🎬 第五页：同桌小方

下课后，小方走过来。

> "你写的字真好看！"小方对 Steve 说。

> "谢谢！你的**山**字也特别棒！"

小方笑了。"你知道'**同**学'为什么有个'同'字吗？"

> "'同'就是我们一起，一起读书，一起写字。"

Steve 点头："而且我们都有同一个**老师**，在同一个**学校**。"

> "对！"小方拿出一块小木牌。

他在木牌上写：

```
   同 学
   
   有同一位老师
   在同一间学校
   读同一本书
   写同一个字
   我们是——同学！
```

Steve 接过木牌，也在上面写：

```
   ✏️ 书 + 包 + 本 + 笔 + 尺
   
   这些都是我的好朋友
   它们陪我上学
   陪我写字
   陪我读书
```

两人把木牌挂在教室墙上。

## 🎬 第六页：放学了

太阳开始下山。方块学校的钟声再次响起——

> "🔔 当当当！放学啦！"

同学们收拾书包。Steve 把书、本子、笔、尺子装进书包。

> "今天读了什么书？"小方问。

> "我读了一本关于海的书！还写了'家'字——它变成了一个真的房子！"

> "我写了'山'，变成了一座小山！"

> "我写了'水'，变成了一条小河！"

老师说："明天我们继续。记得——"

```
   📚 回家也要：
   读一读书
   写一写字
```

大家齐声回答："知道啦！老师再见！"

Steve、Alex 和小方一起走出校门。

> "原来来**学校**上学，是这么开心的一件事。"Steve 说。

> "因为有**老师**，有**同学**，有**书**可以读，有**字**可以写。"Alex 补充。

夕阳下，三个小小的身影走在回家的路上。

```
   开 心 学 校
   
   开心，因为有同学
   开心，因为有书本
   开心，因为每一天都在学习新的东西
```

## 📝 练习

### 一、读一读，连一连

| 字 | 部首 | 意思 |
|----|------|------|
| 校 | 讠（言字旁） | 把字变成声音 |
| 写 | 木（木字旁） | 在纸上留下字 |
| 读 | 冖（秃宝盖） | 木头房子里交流学问 |

```
   校：___校、___园
   写：___字、___书
