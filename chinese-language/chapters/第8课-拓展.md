# 第8课 拓展篇：韵母游乐场

## 📋 学习目标
- 巩固 6 个单韵母：a o e i u ü
- 熟练韵母 + 声调的组合
- 初步体验"声母 + 韵母"的拼读
- 学会两个简单拼读：b-a → ba、m-i → mi

---

六姐妹带 Steve 和 Alex 来到一个神奇的地方——

   🎡 韵母游乐场 🎡
   
   六个游戏区，

```

"今天我们用游戏来复习！"大姐 a 说。

游乐场入口有一块大地图：

```

> "从哪个开始？"Alex 兴奋地问。

> "全部！"Steve 拉着她跑进去。

## 🎬 第二页：a 区 — 大声喊山谷

🔴 a 区是一座回声山谷。站在谷口大喊，回声会跟你玩！

> "规则：喊一个字，山谷回声会重复它，但变一个声调！"

Steve 先来。他深吸一口气，喊：

> "花——huā——！"

回声回来了："花——huà——！"（从第一声变成了第四声！）

> "咦？我的花变成了话？"

Alex 也试试：

> "大——dà——！"

回声："大——dā——！"（从第四声变成第一声！）

> "哈哈哈，山谷在跟我们玩声调游戏！"

```
   🗣️ 山谷游戏规则：
   你喊一个声调 → 山谷回另一个声调
   
   你喊 mā → 回声 má
   你喊 bà → 回声 bā
   
   这是山谷在考你：
   你能分辨不同的声调吗？
```

两人在山谷里玩了好久，一声、二声、三声、四声轮着喊。

## 🎬 第三页：o 区和 e 区 — 摩天轮 + 滑梯

🟡 o 区是一个巨大的摩天轮。每节车厢里有一个带 o 的字。

> "规则：车厢转到最高点，你必须读出里面的字！"

```
   第一圈：我 (wǒ) — ǒ！  火 (huǒ) — uǒ！
   第二圈：左 (zuǒ) — uǒ！ 多 (duō) — uō！
```

Steve 坐着摩天轮升到最高处："我——ǒ！火——uǒ！"

转到最低处："左——uǒ！多——uō！"

---

🔵 e 区是一条弯弯曲曲的滑梯。滑梯上写着带 e 的字。

> "规则：滑到每个弯的时候，读出上面写的字！"

Alex 从顶部滑下——

```
   第一个弯：叶 (yè) — è！
   第二个弯：车 (chē) — ē！
   第三个弯：鹅 (é) — é！
   第四个弯：色 (sè) — è！
```

> "好玩！又读字又滑滑梯！"

## 🎬 第四页：i 区、u 区、ü 区

💚 i 区的碰碰车上有各种形状的拼音卡片。撞到别的车，就要读出对方车上的拼音！

```
   🚗 Steve 的车：bǐ（笔）
   🚙 Alex 的车：dì（地）
   
   碰！→ "bǐ！dì！"
   
   🚗 nǐ（你）
   🚙 xī（西）
   
   碰！→ "nǐ！xī！"
```

---

💜 u 区的小火车穿过山洞。隧道里黑黑的，只有字在发光：

```
   进入隧道——
   🚂 读 (dú) ✨ → "ú！"
   🚂 书 (shū) ✨ → "ū！"
   🚂 不 (bù) ✨ → "ù！"
   🚂 木 (mù) ✨ → "ù！"
   出隧道！
```

---

🧡 ü 区的喷泉会喷出带 ü 的水珠。每一个水珠里有一个字：

```
   💧 鱼 (yú) → ǘ！
   💧 雨 (yǔ) → ǚ！
   💧 绿 (lǜ) → ǜ！
   💧 女 (nǚ) → ǚ！
```

> "用手接住水珠，读出里面的字！"

Steve 和 Alex 在喷泉里跳来跳去，一边接水珠一边念字。

## 🎬 第五页：拼读初体验

六个区都玩完后，六姐妹集合在广场上。

> "你们已经认识了全部韵母。现在来试试——声母 + 韵母 = 完整的拼音！"

> "什么是声母？"Steve 问。

> "声母就像钥匙，韵母就像锁。钥匙插进锁里，转一下——声音就出来了！"

```
   声母 + 韵母 = 拼音
   
   例：b + a = ba
       声母 韵母  拼音
```

> "来，试两个最简单的："

```
   b + a → ba
   
   b (声母) + ā (韵母一声) = bā (八)
   b (声母) + á (韵母二声) = bá (拔)
   b (声母) + ǎ (韵母三声) = bǎ (把)
   b (声母) + à (韵母四声) = bà (爸)
```

Steve 眼睛亮了："所以，b 加上不同声调的 a，就变成不同的字！"

```
   m + i → mi
   
   m (声母) + ī (韵母一声) = mī (眯)
   m (声母) + í (韵母二声) = mí (迷)
   m (声母) + ǐ (韵母三声) = mǐ (米)
   m (声母) + ì (韵母四声) = mì (蜜)
```

Alex 惊叹："一个 m 一个 i，四种声调，四个不同的字！"

> "这就是拼音的魔法——用很少的字母，组合出所有的汉字发音！"

```
   🎯 今日拼读公式：
   
   b + a = ba (八拔把爸)
   m + i = mi (眯迷米蜜)
   p + o = po (坡婆叵破)
   d + e = de (的得地德)
```

## 📝 练习

### 一、找韵母

写出下面字的韵母和声调：

```
   你 (nǐ)  → 韵母：___  声调：___
   书 (shū) → 韵母：___  声调：___
   雨 (yǔ)  → 韵母：___  声调：___
   大 (dà)  → 韵母：___  声调：___
   火 (huǒ) → 韵母：___  声调：___
   叶 (yè)  → 韵母：___  声调：___
```

### 二、拼一拼

```
   b + ā = ___（八）
   b + à = ___（爸）
   m + ǐ = ___（米）
   d + à = ___（大）
   n + ǐ = ___（你）
   l + ǜ = ___（绿）
```

### 三、选韵母填空

```
   a / o / e / i / u / ü

   花 h___ā    火 h___ǒ     叶 y___è
   笔 b___ǐ    书 sh___ū    雨 ___ǔ
   大 d___à    我 w___ǒ     你 n___ǐ
```

### 四、读儿歌

```
   b-a ba，bā bá bǎ bà，
   m-i mi，mī mí mǐ mì，
   d-a da，dā dá dǎ dà，
   n-i ni，nī ní nǐ nì。

   声母是钥匙，韵母是锁，
   钥匙加锁，拼音有我！
```

## 🏆 挑战 — 拼读小达人

**第一关：拼读连连看 🔗**

```
   左边              右边
   b + ā      →       bǐ
   m + ǐ      →       bā
   d + à      →       lǜ
   l + ǜ      →       mǐ
   b + ǐ      →       dà
```

**第二关：同韵母字集合 📦**

把韵母相同的字放一起：

```
   爸 八 大 笔 你 地 书 木 读 雨 女 绿
   
   a 韵母组：___ ___ ___
   i 韵母组：___ ___ ___
   u 韵母组：___ ___ ___
   ü 韵母组：___ ___ ___
```

**第三关：创作拼音歌 🎵**

用 b-a 或 m-i 的四种声调，各编一句话：

```
   bā（八）：我有___个本子。
   bá（拔）：用力___萝卜。
   bǎ（把）：给我一___尺子。
   bà（爸）：我___很高大。
```
