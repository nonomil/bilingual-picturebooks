# 第5课 拓展篇：家族大聚会

## 📋 学习目标
- 巩固第5课 7 个字：爸妈兄妹爷奶宝
- 新增字：**爱 家 亲**（+3字）
- 学会表达"我爱我家"

**累计识字：49字**

---

## 🎬 第一页：村庄的节日

今天村庄很热闹——是一年一度的**家族聚会日**！

每家每户都打开门，摆出好吃的东西。院子里飘着香味。

> "今天是家族的节日！"奶奶笑着。

Steve 看到每家门口挂着一个大牌子：

```
   🏠 爱 (ài) — LOVE
```

> "每家都挂着**爱**字！"Alex 好奇。

奶奶解释："有爱，才是一个家。"

## ✏️ 第二页：爱

**爱** [ài] (10画)

> 上面是"爫"+"冖"，中间是"心"，下面是"夊"——用心去感受。

```
笔画顺序：
①丿(撇)②丶(点)③丶(点)④丿(撇)⑤丶(点)⑥一(横钩)⑦丶(点)⑧一(横钩)⑨丶(点)⑩乛(横撇)

写成口诀：爪字头，包住心，爱在心里暖自身
```

**组词：** 爱心(ài xīn)、我爱(wǒ ài)
**句子：** 我爱我的家。

```
  ✨ 爱 = 用心 + 行动
```

## 🎬 第三页：家是什么

大家围坐在一起。奶奶说：

> "我来说说**家**的意思。"

**家** [jiā] (10画)

> 上面"宀"是房子，下面"豕"是小猪——古时候家里养猪，有房有猪就是家。

```
笔画顺序：
①丶(点)②丶(点)③一(横钩)④一(横)⑤丿(撇)⑥一(弯钩)⑦丿(撇)⑧丿(撇)⑨丿(撇)⑩丶(捺)

写成口诀：宝盖头下有只猪，有家有爱不孤独
```

**组词：** 回家(huí jiā)、一家人(yī jiā rén)
**句子：** 这是我的一家。

```
  🏠 家 = 房子 + 家人 + 爱
```

## 🎬 第四页：亲

爷爷站起来，向大家招手：

> "家人，就是最**亲**的人。"

**亲** [qīn] (9画)

> 上面是"立"，下面是"木"——像一棵树站在旁边，最亲近。

```
笔画顺序：
①丶(点)②一(横)③丶(点)④丿(撇)⑤一(横)⑥一(横)⑦一(竖)⑧丿(撇)⑨丶(捺)

写成口诀：立字站上面，木字在下边，亲人就在你身边
```

**组词：** 亲人(qīn rén)、父亲(fù qīn)、母亲(mǔ qīn)
**句子：** 爸爸妈妈是我最亲的人。

## 🎬 第五页：家族猜谜游戏

吃完东西，大家开始玩游戏！

**游戏1：猜家人** 🎭

小朋友抽一张卡片，表演出来，大家猜是谁：

```
   第一个表演：弯着腰，摸着胡子，慢慢走
   → "是爷爷！"
   
   第二个表演：抱着宝宝，轻声唱歌
   → "是奶奶！"
   
   第三个表演：拿着锅铲，转来转去
   → "是妈妈！"
   
   第四个表演：高高举起双手
   → "是爸爸！"
```

> 大家都猜对啦！

## 🎬 第六页：我爱我的家

游戏结束，Steve 站起来：

> "我也想说说我的家人。"

他拿出一张纸，上面写着：

```
   我爱我的家 🏠❤️
   
   我的爸爸 —— 高大，有力
   我的妈妈 —— 做饭，用心
   我的哥哥 —— 陪我玩，照顾我
   
   爷爷在天上看着我
   奶奶每天对我笑
   
   这就是我的家
   我爱的家
```

Alex 也拿出她写的：

```
   我的家 💕
   
   爸爸 妈妈 哥哥 哥哥 我
   五个人的家
   
   不大 不小 刚好
   有爱 有笑 有宝
```

> 大家安静地听着，然后一起拍手。

奶奶擦了擦眼角："写得真好。"

## 🎬 第七页：全家一起唱歌

天色晚了，篝火升起。所有人围成一圈。

爷爷带头唱起家族歌：

```
   🎵 爸爸爸，妈妈妈，
      哥哥大，妹妹小，
      爷爷笑，奶奶抱，
      宝宝宝，爱我家！
      
      天上的云，地上的花，
      都比不上我的家。
      风再大，雨再下，
      回家就不怕！🎵
```

Steve 和 Alex 跟着一起唱。星空下，火光映着每个人的笑脸。

> "今天真好。"Alex 轻轻说。

> "是啊。"Steve 点头。"家，就是世界上最温暖的地方。"

## 🎯 第八页：家族挑战

### 第一关：生字找家

| 字 | 是什么？ |
|----|----------|
| 爱 | 用心去感受 |
| 家 | 有房子有猪 |
| 亲 | 最近的家人 |

### 第二关：填一填

> 我 ___ 我的家。
> 爸爸是我最 ___ 的人。
> 我们的 ___ 有七个人。

### 第三关：画画

画出你的家（Minecraft 风格！），标出每个人：

```
   [你的家]
   这个是 ______ (爸/妈/兄/妹/爷/奶/宝)
   这个是 ______ …
```

### 第四关：写给家人

选一个家人，写一句话：

> 我最爱的家人是 ______。
> 因为 ________________。
