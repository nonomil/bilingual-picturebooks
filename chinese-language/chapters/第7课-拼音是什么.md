# 第7课 拼音是什么

## 📋 学习目标
- 认识"拼音"——汉字的发音魔法
- 学会 3 个单韵母：**a o e**
- 掌握四声调号：**ˉ ˊ ˇ ˋ**
- 用拼音读学过的汉字

**拼音阶段不引入新汉字，用已学 58 字做拼读练习**

## 🎬 第一页：图书馆的秘密

方块学校的图书馆里，Steve 发现了一本奇怪的书。

这本书的封面上画着发光的符文：

```
   ✨ 拼 音 ✨
```

> "这是什么书？"Steve 翻开第一页。

书页突然发光——三个彩色的符文从书中跳了出来：

```
   🔴 A  🟡 O  🔵 E
```

它们在空中旋转、闪烁，像三只小精灵。

> "哇！"Alex 跑过来。"这些是什么？"

书页上浮现出一行字：

```
   拼 音 (pīn yīn)
   
   拼音 = 汉字的发音魔法
   每个汉字都有自己的拼音
   学会了拼音，你就能读出任何汉字！
```

> "这就是**拼音**！"老师说，不知什么时候站在了身后。

> "拼音就像一座桥——从你认识的字，走到你不认识的字。"

```
   🤔 为什么要学拼音？
   
   认识字 → 直接读
   不认识字 → 看拼音 → 读出来！
   
   比如：你看到了「花」
   → 花 = huā
   → 认识「花」就直接读
   → 不认识就看拼音 huā → 读出来！
```

## 🎬 第二页：单韵母 — 拼音家族的元老

书页翻到第二章，标题写着：

```
   单 韵 母 (dān yùn mǔ)
   
   单 = 单独、一个
   韵 = 好听的声音
   母 = 妈妈
   
   单韵母 = 拼音世界里最基础的"声音妈妈"
   一共 6 个：a o e i u ü
```

> "今天我们先认识三位声音妈妈：**a、o、e**。"

三个符文飘到 Steve 面前：


```
   🎵 单韵母口诀：

   扁扁嘴巴 e e e  （像大白鹅叫：鹅——）
```

Steve 跟着念："ɑ——o——e——"

他的声音像魔法一样，书页上浮现出对应的图画：

```
   ɑ → 👄 张开大嘴
   o → ⭕ 圆圆的口
   e → ↔️ 扁扁的嘴
```

## 🎬 第四页：认识四声

书页上出现了一座小山：

```
        1ˉ
       /  \
      /    \
     / 2ˊ   \ 4ˋ
    /        \
   /    3ˇ    \___
  /              
```

> "这座山叫做**声调山**！"老师说。

> "每个拼音都可以爬这座山——这就是**四声**。"

书页展开：

```
   🏔️ 四声调号 (sì shēng diào hào)
   
   第一声 ˉ  —  平平的，像开车走平路  →  ā
   第二声 ˊ  —  往上走，像爬山         →  á
   第三声 ˇ  —  先下后上，像过山车     →  ǎ
   第四声 ˋ  —  往下走，像下山         →  à
```

> "来，跟着 a 一起爬声调山！"

```
   ɑ 的四声：
   
   ā  —  āāā   （平路开车：啊——）
   á  —  ááá   （上坡：啊？什么？）
   ǎ  —  ǎǎǎ   （过山车：啊！好刺激！）
   à  —  ààà   （下坡：啊！我知道了！）
```

Steve 跟着念，手指在空中画声调符号。每画一个，书页就亮起来。

```
   ✏️ 声调符号口诀：
   
   一声平 ˉ  —  高而平
   二声扬 ˊ  —  由低升高
   三声拐弯 ˇ —  先降后升
   四声降 ˋ  —  由高降低
```

## 🎬 第五页：o — 圆圆嘴巴

黄色符文 **o** 滚到 Alex 手心。

```
   o 的读法：
   
   ① 嘴巴圆圆的
   ② 舌头往后缩
```

> "像公鸡打鸣：喔喔喔——"

```
   o 的四声：
   
   ō  —  ōōō   （平声）

   ǒ  —  ǒǒǒ   （拐弯声）
   ò  —  òòò   （降声：哦！明白了！）
```

## 🎬 第六页：e — 扁扁嘴巴

蓝色符文 **e** 轻轻飘到两人中间。

```
   e 的读法：
   
   ① 嘴巴扁扁的，向两边咧开
   ② 舌头往后缩
   ③ 发出声音：鹅——
```

> "像大白鹅：鹅鹅鹅——"

两人一起："鹅——"

蓝光闪烁：

```
   车 (chē)?  — 但是「车」还没学过！
   鹅 (é)     — 鹅！
   叶 (yè)    — 叶子！
```

> "等一下——"Alex 发现了什么。"'鹅'的声调是ˊ（二声），'叶'是ˋ（四声）！"

```
   e 的四声：
   
   ē  —  ēēē   （平声）
   é  —  ééé   （扬声：鹅！大白鹅！）
   ě  —  ěěě   （拐弯声：恶心）
   è  —  èèè   （降声：饿！我饿了！）
```

> "对了！同样的 e，不同的声调，意思完全不同！"蓝色符文说。

```
   é = 鹅  🦢  大白鹅
   è = 饿  🍽️ 肚子饿
```

---

> 【标A: 语文课标一上·汉语拼音·声母韵母整体认读】

### ❌常见误解

| ❌ 错误发音/写法 | ✅ 正确发音/写法 |
|-------|-------|
| b和d分不清（"b像6，d反6"） | b=右耳b，圆圈在右；d=左耳d，圆圈在左 |
| p和q分不清（"p像9，q反9"） | p=泼水p，圆圈在右向下；q=气球q，圆圈在左向下 |
| zh/ch/sh 读成 j/q/x | 舌尖翘起（翘舌音），不是舌面贴硬腭 |
| 前鼻音和后鼻音不分 | an≠ang，舌头位置不同（前鼻音舌尖顶上齿，后鼻音舌根抬起） |

🧠 想一想
1. **观察推理**：b、p、m、f 都用嘴唇发音，d、t、n、l 用舌尖。你还能找到其他发音位置的规律吗？
2. **反事实**：如果拼音里没有声调（一、二、三、四声），"妈妈骂马"这4个字还能分得清吗？

## 📊 本课小结

拼音三大基础：
- [ ] 拼音是什么：汉字的发音魔法，帮你读出不认识的字
- [ ] 单韵母 a o e：张大 a、圆圆 o、扁扁 e
- [ ] 四声调号：一声平ˉ、二声扬ˊ、三声拐弯ˇ、四声降ˋ

四声口诀：
- [ ] ā á ǎ à — a 的四声
- [ ] ō ó ǒ ò — o 的四声
- [ ] ē é ě è — e 的四声

> **已学单韵母：a o e（3/6）**
> 下次学习：i u ü
