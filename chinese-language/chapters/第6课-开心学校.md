# 第6课 开心学校

## 📋 学习目标
- 认识 9 个新字：**老 师 同 学 书 包 本 笔 尺**
- 学会介绍学校和学习用品
- 掌握每个字的笔画顺序

**累计识字：55字**（前课46 + 本课9）

## 🎬 第一页：方块学校开学了

太阳从方块山后升起，村庄里响起了钟声——

> "🔔 当当当！"

Steve 揉揉眼睛："什么声音？"

> "是**学校**的钟声！"Alex 兴奋地跳起来。"方块学校今天开学啦！"

他们跑到村口，看到一座用各色方块搭成的建筑。门口挂着一块大木牌，上面写着：

```
   📚 开心学校 📚
```

一位胡子花白的村民站在门口，笑眯眯的。

> "欢迎来到开心学校！我是你们的**老师**。"

> "**老**师好！"Steve 和 Alex 鞠躬。

```
   老 (lǎo) — 年纪大、有经验
   师 (shī) — 教别人的人
   老师 = 教我们知识的人
```

## ✏️ 第二页：老 + 师

老师带着他们走进教室。教室里有木头桌子和方块椅子，墙上画着彩色图案。

**老** [lǎo] (6画)

> 上面是"耂"（老字头），下面是"匕"——年纪大的人拄着拐杖。

```
笔画顺序：
①一(横)②一(竖)③一(横)④丿(撇)⑤丿(撇)⑥一(竖弯钩)

写成口诀：老字头上横竖横，一撇再撇拐杖撑
```

**组词：** 老师(lǎo shī)、老人(lǎo rén)
**句子：** 老师是个老人。

---

**师** [shī] (6画)

> 左边是"刂"（刀字旁简化），右边是"帀"——教人本领的人。

```
笔画顺序：
①一(竖)②丿(撇)③一(横)④一(竖)⑤一(横折钩)⑥一(竖)

写成口诀：左边一竖加一撇，右边横竖折钩收
```

**组词：** 老师(lǎo shī)、师父(shī fu)
**句子：** 老师教我写字。

```
   ✨ 老 + 师 = 老师
   有经验的、教人本领的人 = 老师
```

## ✏️ 第三页：同 + 学

教室里已经坐着几个小朋友。

> "这是你们的**同学**！"老师说。

**同** [tóng] (6画)

> 外面是"冂"（包住），里面是"一"+"口"——一起的意思。

```
笔画顺序：
①一(竖)②一(横折钩)③一(横)④一(竖)⑤一(横折)⑥一(横)

写成口诀：包字框加一口，大家一起做朋友
```

**组词：** 同学(tóng xué)、一同(yī tóng)
**句子：** 我和你是同学。

---

**学** [xué] (8画)

> 上面是"⺍"（学字头），中间是"冖"（秃宝盖），下面是"子"——小孩在学习。

```
笔画顺序：
①丶(点)②丶(点)③丿(撇)④丶(点)⑤一(横钩)⑥一(横钩)⑦一(竖钩)⑧一(横)

写成口诀：三点一撇秃宝盖，下面一个子字来
```

**组词：** 学习(xué xí)、学校(xué xiào)
**句子：** 我去学校上学。

```
   ✨ 同 + 学 = 同学
   一起学习的人 = 同学
```

一位小朋友站起来："你好！我叫小方。从今天起，我们就是**同学**了！"

## ✏️ 第四页：书

老师走到讲台前，举起一件东西。

> "你们知道这是什么吗？"

> "**书**！"大家齐声说。

**书** [shū] (4画)

> 横折钩作架子，中间一竖作书脊，加上一点代表字。

```
笔画顺序：
①一(横折)②一(横折钩)③一(竖)④丶(点)

写成口诀：横折两笔加竖点，小小书本在眼前
```

**组词：** 书本(shū běn)、看书(kàn shū)
**句子：** 我有一本好书。

老师说："书，是我们的好朋友。里面有故事、有知识、有整个世界！"

他打开书，里面画满了方块图案：方块山、方块河、方块动物……

```
   📖 书里有什么？
   有字、有画、有故事
   有整个世界！
```

## ✏️ 第五页：包 + 本

Steve 拿出自己的东西。

> "这是我的**书包**！"他说。

**包** [bāo] (5画)

> 外面是"勹"（包字框），里面是"巳"——把东西包起来。

```
笔画顺序：
①丿(撇)②一(横折钩)③一(横折)④一(横)⑤一(竖弯钩)

写成口诀：包字框加口字心，背在肩上上学去
```

**组词：** 书包(shū bāo)、包子(bāo zi)
**句子：** 我的书包里有书。

---

Steve 从书包里掏出一个小本子。

> "这是**本**子！用来写字的。"

**本** [běn] (5画)

> "木"字下面加一横——树的根，引申为"根源、本子"。

```
笔画顺序：
①一(横)②一(竖)③丿(撇)④丶(捺)⑤一(横)

写成口诀：木字写好后，下边一横收
```

**组词：** 本子(běn zi)、书本(shū běn)
**句子：** 本子上写字。

```
   🎒 书包 + 本子
   书包里装着：
   📖 书 📓 本子 ✏️ 笔 📏 尺子
```

## ✏️ 第六页：笔 + 尺

Alex 也打开她的书包。

> "我带了**笔**和**尺**子！"

**笔** [bǐ] (10画)

> 上面是"⺮"（竹字头），下面是"毛"——竹子做的写字工具。

```
笔画顺序：
①丿(撇)②一(横)③丶(点)④丿(撇)⑤一(横)⑥丶(点)⑦丿(撇)⑧一(横)⑨一(横)⑩一(竖弯钩)

写成口诀：竹字头在上，毛字在下方，写字画画都用它
```

**组词：** 毛笔(máo bǐ)、笔尺(bǐ chǐ)
**句子：** 我用笔写字。

---

**尺** [chǐ] (4画)

> 像一个人伸开手臂量东西的样子——用来测量。

```
笔画顺序：
①一(横折)②一(横)③丿(撇)④乀(捺)

写成口诀：横折一横撇加捺，量一量有多长
```

**组词：** 尺子(chǐ zi)、尺寸(chǐ cun)
**句子：** 尺子用来量长度。

> "用**笔**写字，用**尺**画线！"

```
   ✏️ 笔 — 写字、画画
   📏 尺 — 量长度、画直线
```

## 🎬 第八页：故事时间 — 小铅笔的冒险

放学后，Steve 的铅笔掉在地上。

"咦，我的笔呢？"他四处找。

> "我在这里！"一个细小的声音。

铅笔从地上蹦起来："谢谢你每天拿我写字！我想看看这个世界。"

铅笔跳上书桌，在纸上画了一条线——

> "这是我画的**直线**！用**尺**子画的！"

铅笔又跳到窗外，在泥地上写字：

```
   开心学校
   老师好
   同学好
```

小鸟们围过来看："哇，这是谁写的？真好看！"

铅笔骄傲地说："是**我**——一支小**笔**！"

Steve 笑着把铅笔捡起来："好了，回家吧。明天还要**上学**呢。"

### 二、填一填

Steve 第一天上学，他在日记里写：

> 今天我去 ___（学校/回家）。

> 我背着 ___（书包/尺子）。
> 里面有一本 ___（书/笔）、一个 ___（本子/尺子）、一支 ___（笔/包）、一把 ___（尺/书）。

### ❌常见误解

| ❌ 错误写法/理解 | ✅ 正确写法/理解 |
|-------|-------|
| "日"写成"目"（中间多一横） | 日=太阳，中间一横，不是两横 |
| "山"写成三竖一样高 | 中间一竖最高，两边的低 |
| "水"的笔画随便写 | 笔顺：竖钩 → 横撇 → 撇 → 捺 |
| 把象形字当画看，不记字形 | 象形字是"从画变来的字"，要记住现在的样子 |

🧠 想一想
1. **观察推理**：为什么"日"里面只有一横而不是两横？（提示：太阳只有一个）
2. **反事实**：如果古人把"山"画成三座一样高的山峰，现在的"山"字会是什么样子？
