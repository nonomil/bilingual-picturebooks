## 🎬 第二页：静夜思 — 李白

花园第一区——月光下的庭院。刻在石碑上的是中国最著名的诗：

```
   🌙 《静夜思》 [唐] 李白
   
   床前明月光，
   疑是地上霜。
   举头望明月，
   低头思故乡。
```

**新字学习：**

```
   床 [chuáng] (7画) — 睡觉的床
   光 [guāng] (6画) — 光线、光亮
   疑 [yí] (14画) — 怀疑、好像
   举 [jǔ] (9画) — 举起、抬起
   望 [wàng] (11画) — 向远处看
   低 [dī] (7画) — 向下（复习）
   故 [gù] (9画) — 原来的、旧的 → 故乡 = 老家
   乡 [xiāng] (3画) — 家乡、乡村
```

**诗意解读：**

> 诗人李白独自在异乡。半夜醒来，看到床前洒满了明亮的月光。月光太白太亮了，他差点以为地上结了霜。他抬起头，看着天上的月亮。又低下头，想起了远方的家乡。

> "一千多年前，一个人看着月亮想家。今天，我们读他的诗，依然能感受到他的思念。这就是诗歌的力量。"

```
   🎵 静夜思 — 节奏朗读：
   床前——明月——光——
   疑是——地上——霜——
   举头——望明——月——
   低头——思故——乡——
```

Steve静静地看着天上的月亮——方块世界的月亮又大又方。他好像也想起了什么。

## 🎬 第三页：咏鹅 — 骆宾王

花园第二区——白鹅池塘。水面上游着几只大白鹅。

```
   🦢 《咏鹅》 [唐] 骆宾王
   
   鹅，鹅，鹅，
   曲项向天歌。
   白毛浮绿水，
   红掌拨清波。
```

**新字学习：**

```
   鹅 [é] (12画) — 大白鹅
   曲 [qū] (6画) — 弯曲
   项 [xiàng] (9画) — 脖子
   浮 [fú] (10画) — 浮在水面
   拨 [bō] (8画) — 拨开、拨动
```

**诗意解读：**

> 一个七岁的小孩子写了一首诗——他看到池塘里的白鹅，鹅弯着脖子朝天叫，白色的羽毛漂浮在绿色的水面上，红色的脚掌拨动清清的水波。简单四句话，写出了白鹅的样子、声音、颜色、动作。

> "据说骆宾王写这首诗时才七岁——跟你差不多大！所以你也可以写诗。"

```
   🦢 颜色之美：
   白毛（白色）+ 绿水（绿色）
   红掌（红色）+ 清波（透明）
   
   四句诗，四种颜色，一幅画！
```

## 🎬 第四页：春晓 — 孟浩然

花园第三区——春天的早晨。鸟语花香，落花满地。

```
   🌸 《春晓》 [唐] 孟浩然
   
   春眠不觉晓，
   处处闻啼鸟。
   夜来风雨声，
   花落知多少。
```

**新字学习：**

```
   眠 [mián] (10画) — 睡眠
   晓 [xiǎo] (10画) — 天亮、早晨
   啼 [tí] (12画) — 鸟叫
   落 [luò] (12画) — 落下
```

**诗意解读：**

> 春天的早晨，人睡得太香了，天亮了都不知道。醒来后，到处都能听到鸟儿的叫声。想起来昨夜有风声雨声——不知道花儿被风雨打落了多少？

> "四句话，从睡觉写到醒来，从鸟叫想到落花——把一个春天的早晨写得完整又美丽。"

```
   🌸 春晓的节奏：
   春眠——不觉——晓——
   处处——闻啼——鸟——
   夜来——风雨——声——
   花落——知多——少——
```

Alex嗅了嗅花园里的花香："春眠不觉晓——春天的早上真的不想起床呢！"

## 🎬 第五页：我也是小诗人

花园最后是一面空白的诗墙——上面写着："写下你的诗"。

> "你已经认识了一百多个汉字——现在试试用它们写一首诗吧。不用很长，四句就够了。"

Steve写下：

```
   方块小诗
   
   方块天，方块地，
   方块朋友在一起。
   方块太阳方块月，
   方块世界真美丽。
```

Alex写下：

```
   颜色诗
   
   红花开了，
   白云飘了，
   蓝天远了，
   我的朋友笑了。
```

即使字还不多，即使诗句还很简单——这就是诗歌的开始。

> "写诗不需要很多字。需要的是——你想说什么。"

```
   🎵 古诗小结 🎵
   
   静夜思 — 看月亮想家（李白）
   咏鹅 — 白鹅在水里（骆宾王，7岁写的！）
   春晓 — 春天的早晨（孟浩然）
```
