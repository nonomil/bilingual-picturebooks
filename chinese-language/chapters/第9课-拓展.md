# 第9课 拓展篇：声母大练兵

## 📋 学习目标
- 巩固 8 个声母：b p m f d t n l
- 熟练拼读技能：声母 + 单韵母 + 声调
- 引入"双拼音节"概念
- 用拼读解读已学汉字

## 🎬 第二页：第一关 — 嘴唇特训

b p m f 四位嘴唇兄弟站在第一个训练区。

> "规则：听我们发音，立刻说出是哪个声母！还要说出对应的韵母！"

第一轮——b vs p：

```
   p 精灵发音，配韵母 a
   → "pa——一声！"
   
   Steve："声母是 p，韵母是 a，声调是一声——pā！"
   ✓ 正确！
   
   b 精灵发音，配韵母 i
   → "bi——三声！"
   
   Alex："声母是 b，韵母是 i，声调是三声——bǐ！"
   ✓ 正确！正是「笔」！
```

第二轮——m 和 f：

```
   m 精灵："mā——我的声母是 m，韵母 a，一声——妈妈！"
   f 精灵："fù——我的声母是 f，韵母 u，四声——父亲！"
```

> "你们已经能听出送气和不送气的区别了！"

```
   🎯 b/p 分辨秘诀：
   手心放嘴前
   有气 p / 无气 b
   
   🎯 m/f 分辨秘诀：
   闭嘴唇 m / 咬下唇 f
```

## 🎬 第三页：第二关 — 舌尖挑战

> "规则：仔细听，快速判断！特别是 n 和 l！"

```
   声音1：dà —— 声母？韵母？声调？
   Alex："声母 d！韵母 a！四声！大！"
   ✓
   
   声音2：tā —— 声母？韵母？声调？
   Steve："声母 t！韵母 a！一声！他！"
   ✓
   
   声音3：nǐ —— 声母？韵母？声调？
   Alex："声母 n！韵母 i！三声！你！"
   ✓
   
   声音4：lǜ —— 声母？韵母？声调？
   Steve："声母 l！韵母 ü！四声！绿！"

```

   发不出来的是 n（鼻子出气）

   ✓
   
   字：大
   
   Alex："大 = dà。声母是 d，韵母是 a，声调是四声！"
   ✓

> "b 将军，声母和韵母——为什么一定要在一起？"

```
   🌉 拼音桥 🌉
   
   桥的一头是"认识的字"🏠
   桥的另一头是"不认识的字"🗺️
```

> "这座桥叫拼音桥。桥上的每一块砖，都是一个音节——一个声母加一个韵母。"

> "当你认识一个字时，你可以直接走过去。但当你不认识时——"

b 将军指着桥上的砖：

```
   🧱 b + a = ba → 爸（认识！直接读）
   🧱 p + a = pa → 爬（看拼音，读出来！）
   🧱 n + i = ni → 你（认识！）
   🧱 l + a = la → 拉（看拼音，读出来！）
```

> "拼音就是这座桥——帮你从认识的字，走到不认识的字。"

Steve 懂了："所以学拼音，就是往桥上铺更多的砖！"

> "没错！今天铺了 b p m f d t n l + a o e i u ü，你已经有了几十块砖！"

> "以后每学一个新的声母和韵母，你的桥就长一截，能去的地方就更多！"

Alex 站在桥上往远处看："那边的世界好大！"

> "继续学拼音吧！总有一天，你能走到任何你想去的地方——读到任何你想读的书。"

## 📝 练习

### 一、拆一拆

把下面的字拆成声母和韵母：

```
   字      拼音    声母    韵母    声调
   爸      bà      ___     ___     ___
   妈      mā      ___     ___     ___
   你      nǐ      ___     ___     ___
   大      dà      ___     ___     ___
   他      tā      ___     ___     ___
   发      fā      ___     ___     ___
   绿      lǜ      ___     ___     ___
   爬      pá      ___     ___     ___
```

### 二、拼一拼

声母 + 韵母 + 声调 → 完整的拼音：

```
   b + à = ___       p + á = ___
   m + ā = ___       f + ā = ___
   d + à = ___       t + ā = ___
   n + ǚ = ___       l + ǜ = ___
```

### 三、听音辨声母

让大人念下面的拼音，你写出声母：

```
   1. ___ā  → 可能声母：b/p/m/f  → 听是哪个？
   2. ___à  → 可能声母：d/t/n/l  → 听是哪个？
   3. ___ǐ  → 可能声母：b/p/m/f/d/t/n/l → 听是哪个？
```

### 四、拼音桥

画一座桥，在桥的砖块上写出你今天会拼的拼音：

```
   🌉 我的拼音桥
   
   已经铺设的砖块：
   bā bà pā pá mā mǐ fā fù
   dà tā nǐ nǚ lā lǜ
   
   我能用拼音读出的字有：
   _______________
```
