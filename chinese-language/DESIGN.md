## 二、整体设计规范

### 基础风格（沿袭数学绘本）

| 维度 | 规格 |
|------|------|
| 叙事线 | Steve & Alex 在方块世界冒险，解决一个个问题 |
| 每课结构 | 学习目标 → 故事展开 → 教学核心 → 练习 → 挑战 → 小结 |
| 拓展篇 | 每课后跟拓展篇，巩固练习 + 深入内容 |
| 图片风格 | Minecraft pixel art / 方块风格（数学绘本风格复制） |
| 图片路径 | `./img/chapter-N/page-NN.png` |
| 每课图数 | 基础篇 8-12 张，拓展篇 6-10 张 |
| 读者年龄 | 语文：5-7 岁（幼小衔接），英语：4-6 岁（零基础） |
| 交互元素 | emoji 表情、对话气泡、游戏化挑战环节 |

### 角色延续

- **Steve** — 主角，从零开始学习
- **Alex** — 引导者，负责解释概念
- **Bob** — 村民，辅助角色

## 三、语文绘本 — 课程体系

### 总览：12 课基础篇 + 12 课拓展篇（目标识字量：~96字）

```
第1课  神奇汉字     → 象形字入门（日月山水火木田石）  ➕ 8字
第2课  笔画小课堂   → 横竖撇捺点提折钩（一丨人大小） ➕ 9字
第3课  汉字小天地   → 天地人你我他上下左右中          ➕ 10字
第4课  美丽大自然   → 云雨风雪花鸟虫鱼牛羊马          ➕ 10字
第5课  我爱我家     → 爸妈我兄妹爷奶宝宝              ➕ 8字
第6课  拼音王国Ⅰ   → 单韵母aoeiuü+四声（无新汉字）   ➕ 0字
第7课  拼音王国Ⅱ   → 声母bpmfdtnlgkh （无新汉字）    ➕ 0字
第8课  数字与颜色   → 一二三四五十百千万红黄蓝绿      ➕ 14字
第9课  我的身体     → 口耳目手足头牙心                ➕ 8字
第10课 快乐的一天   → 吃喝看走笑来去坐睡站            ➕ 10字
第11课 动物大联欢   → 狗猫兔鸟鱼虫                     ➕ 6字
第12课 我们的世界   → 复习+短句阅读（累计约96字）      ➕ 10字
```

**字表总览（~96字，对标小羊上山L1 + 部编版第1单元）：**
```
数字：一二三四五十百千万
自然：日月山水火木田石天地云雨风雪星
人伦：爸妈我兄妹爷奶你我他
身体：口耳目手足头牙心
动物：鸟虫鱼牛羊马狗猫兔
动作：吃喝看走笑来去坐睡站飞跑跳
方位：上下左右中
颜色：红黄蓝绿白黑
其他：大小多少花早
```

### 课程设计原则

1. **字序：** 从象形字→笔画→简单独体字→复合字
2. **每课新字：** 6-12个，课文密集出现+练习巩固
3. **已有字复用：** 后课至少含60%前课字，混
